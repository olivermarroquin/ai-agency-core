This repository contains reusable components used across all ventures.

It is the foundation layer for:

- prompts
- templates
- workflows
- standards

## Role in System

second-brain → thinking

ai-agency-core → reusable building blocks

## Key Principle

Everything here must be reusable across multiple projects.

If something is project-specific, it does not belong here.
