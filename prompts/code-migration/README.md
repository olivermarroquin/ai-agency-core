# Code Migration Prompts

## Purpose

These prompts support the code-migration workflow.

They are reusable prompt templates for:

- analyzer
- planner
- coder
- reviewer

## Role in System

- ai-factory = workflow execution and orchestration
- ai-agency-core = reusable prompts and standards

## Rule

These prompts should remain generic enough to reuse across multiple projects, while still enforcing narrow, safe migration behavior.
